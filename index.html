End to end ML application

Custom Object Detection

The project aims to leverage the power of transfer learning by utilizing the TensorFlow 2 Object Detection API and the TensorFlow 2 Model Zoo models to accurately predict four different types of birds. By building upon existing pre-trained models and fine-tuning them with bird-specific data, the project seeks to achieve high accuracy and efficiency in bird classification, which can have numerous applications in fields such as ecology, conservation, and wildlife management.

Enviornmental Sounds Classification

The project utilizes the Sounds8K dataset to train a machine learning model capable of classifying 10 different classes of environmental sounds. The trained model has practical applications, including enhancing emergency services' responsiveness and enabling sophisticated threat analysis by security agencies. The model can detect criminal and anti-social behavior in real-time, contributing to safer communities.

A few more Projects

Following are a few more projects that I have worked on. Feel free to visit my GitHub account to learn more about these projects.

Higgs Boson Classifier

A machine learning model utilizing CUDA libraries such as cuDF, cuML, and cuPy, trained on LHC dataset, employing XGBoost and Random Forest algorithms on GPUs to classify Higgs boson signal events from background events.

Bird Skeletal Analysis

The objective of this project is to create a machine learning model that can predict the living habits of birds based on their bone measurements. To achieve this objective, three different machine learning models were trained: Logistic Regression, Support Vector Machine (SVM) and K-Nearest Neighbors (KNN).

House Sale Prediction

This ML project aims to predict house prices in King County, including Seattle, using a dataset of homes sold between May 2014 and May 2015.

Logistic Regression

This project uses logistic regression to train on two different datasets: the Titanic dataset to predict passenger survival based on features, and the Fake Advertising dataset to predict ad clicks based on user features. The goal is to build accurate classification models to inform decision-making in each scenario.

Flask Application

This project is a Python Flask web application that features multiple webpages built using HTML, CSS, and Jinja templating. The webpages are served through defined routes in the Python app, which utilizes Flask libraries to create a seamless user experience.
