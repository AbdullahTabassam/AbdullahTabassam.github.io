MLOps

End to end ML application for object Detection

In this project, we have developed a Flask application that performs object detection using TensorFlow 2 (TF2) object detection models. The application's user interface is designed using HTML and CSS files, providing a simple and interactive experience. The TF2 models are served on TF2 serving for efficient deployment and management, and the entire project is containerized using Docker for easy scalability.

Bird Species Detection

The project aims to leverage the power of transfer learning by utilizing the TensorFlow 2 Object Detection API and the TensorFlow 2 Model Zoo models to accurately predict four different types of birds. By building upon existing pre-trained models and fine-tuning them with bird-specific data, the project seeks to achieve high accuracy and efficiency in bird classification, which can have numerous applications in fields such as ecology, conservation, and wildlife management.

Sign Language Detection

This project features a custom TF2 object detection model based on ResNet101 Faster R-CNN algorithm. It effectively identifies all 26 English alphabet gestures in sign language, trained on a comprehensive dataset available for download from Roboflow. The model aims to enhance communication accessibility for the hearing-impaired community.

A few more Projects

Following are a few more projects that I have worked on. Feel free to visit my GitHub account to learn more about these projects.

Volume Control using Hand gestures

This project showcases a Python script that utilizes the MediaPipe and OpenCV libraries to detect hand gestures from webcam input. With the ability to recognize thumb and index finger movements, the script controls the volume of a Windows computer using the PyCaw library. Enhance your interaction with technology by adjusting volume levels through intuitive hand gestures.

Bird Skeletal Analysis

The objective of this project is to create a machine learning model that can predict the living habits of birds based on their bone measurements. To achieve this objective, three different machine learning models were trained: Logistic Regression, Support Vector Machine (SVM) and K-Nearest Neighbors (KNN).

Higgs Boson Classifier

A machine learning model utilizing CUDA libraries such as cuDF, cuML, and cuPy, trained on LHC dataset, employing XGBoost and Random Forest algorithms on GPUs to classify Higgs boson signal events from background events.

Health Coach GPT (LLM)

This project features a language chain model that leverages the OpenAI API key to create a powerful language model. The model is specifically designed to generate personalized diet and workout plans for users. It is integrated into a Flask app and a website, providing an interactive and user-friendly experience.

Enviornmental Sounds Classification

The project utilizes the Sounds8K dataset to train a machine learning model capable of classifying 10 different classes of environmental sounds. The trained model has practical applications, including enhancing emergency services' responsiveness and enabling sophisticated threat analysis by security agencies. The model can detect criminal and anti-social behavior in real-time, contributing to safer communities.

NLP Ham and Spam SMS detection

This project utilizes NLP techniques to classify text messages as spam or ham. By training a Random Forest classifier on a labeled dataset, it employs preprocessing steps such as tokenization and TF-IDF vectorization. The Jupyter Notebook provides step-by-step instructions, evaluation metrics, and optimization using GridSearchCV.
